# BetterBrain

Intelligent workflow automation that understands your business.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: GSAP + ScrollTrigger
- **Language**: TypeScript
- **Deployment**: Railway

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

- `/` — Homepage
- `/solutions` — Solutions detail
- `/case-studies` — Case studies hub + individual pages
- `/insights` — Thought leadership / blog
- `/about` — About BetterBrain
- `/get-started` — Consultation form

## Deployment

Configured for Railway via `railway.json`. Connect the GitHub repo to Railway and it will auto-deploy on push.
